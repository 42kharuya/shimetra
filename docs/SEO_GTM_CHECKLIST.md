# SEO / GTM Checklist（最低限）

MVPで「やりすぎない」ための最小チェックです。

## 1) 配布導線（GTM）

- [ ] 主要チャネルを最大2つに絞った（[docs/GROWTH_PLAYBOOK.md](GROWTH_PLAYBOOK.md)）
- [ ] UTMの命名ルールを決めた（例: source / medium / campaign）
- [ ] LPのCTAが1つに絞れている
- [ ] 7日分の投稿計画がある

## 2) SEO / OGP（できる範囲で）

- [ ] タイトル/説明（meta）がある
- [ ] OGP画像・説明がある（SNSで崩れない）
- [ ] noindex の扱い（stagingだけnoindex等）を決めた
- [ ] 主要ページがモバイルで読める
- [ ] パフォーマンスが致命的に遅くない

## 3) 計測（最低限）

- [ ] `signup` / `activation` がイベントで取れている（[docs/ANALYTICS_SPEC.md](ANALYTICS_SPEC.md)）
- [ ] UTMをイベントに紐づけられる（可能なら）
