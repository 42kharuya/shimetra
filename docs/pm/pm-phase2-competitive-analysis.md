# 〆トラ Competitive Analysis（pm-phase2）

前提: phase1では「大学生が本当にやりたいことを見つけ、関連する企業候補や次のアクションを提案され、行動後の振り返りをもとに提案が改善される」体験を仮説とした。ここでは公開情報ベースで、近い競合と代替手段を整理する。価格や詳細が公開情報だけで断定できないものは `不明` とする。現時点では、BaseMeが最も直接的に近い競合であり、「市場が空いている」前提では進めない方がよい。

## 勝てる条件

1. 自己理解の結果から、候補企業と次の一手が短時間で具体化されること。
2. 自前の構造化企業DBにより、「なぜこの企業/行動を勧めるのか」を説明できること。
3. BaseMeのような既存AI支援と比べても、「次の一手の具体性」か「提案改善の納得感」で明確に優れること。

## 避けるべき土俵

1. ONE CAREERやマイナビのような情報量・掲載社数の勝負。
2. OfferBoxやdodaキャンパスのような送客量・スカウト量の勝負。
3. Matcherやビズリーチ・キャンパスのような人的ネットワーク量そのものの勝負。
