# AGENTS.md — 共通原則

このファイルはリポジトリ全体に適用される**共通原則**です。
安全性・変更方針・検証姿勢など、すべての AI エージェント/Copilot 操作の土台になります。

## ソースオブトゥルース（優先順位）

| 対象       | 正となるファイル                               |
| :--------- | :--------------------------------------------- |
| 仕様       | `docs/PRD.md`                                  |
| 設計       | `docs/ARCHITECTURE.md`                         |
| 開発ルール | `.github/copilot-instructions.md` / 本ファイル |

矛盾があれば、勝手に解釈せず、必ず質問する。

## 禁止事項

- 著作権的に問題になり得るコード/文章の丸写し（既存サービス模倣のコピペ）をしない
- 不確実な情報を断定しない（必要なら「推測」と明記する）
- 秘密情報（APIキー等）や個人情報をコード・コミット・出力に含めない

## 変更・安全方針

- 影響の大きい変更は段階的に実装し、ロールバック手順を添える
- 変更したら、関連ドキュメント（README / 設計 / チェックリスト等）の更新要否を必ず確認する
- 既存のテストを落とすような修正を含めない

## 検証姿勢

- エラー時の挙動（空状態/権限/通信失敗）を最低限用意する
- クリティカル導線（登録→初回成功体験→課金/申込など）に限って、最小のテスト計画またはテストコードを用意する
- ログ/計測が必要なら「何を・いつ・どこで」記録するかを提案する
